Kaizen

This project was created using HTML, CSS, and Javascript. It was created using React.js and Redux. It is hosted on an express server and connected to a MongoDB database. Kaizen allows you to create processes and improve them over time by letting you log your improvement experiments.

Timer App

This application was created using the Vue.js library. The purpose of timer app is for people to keep close attention to the passing on time.

Wikipedia Viewer

The Wikipedia Viewer was inspired by the great Wikipedia API. It is meant to display images from related articles so that you can better choose what to read.

Focus Gym

Focus Gym was inspired by distractions. Focus Gym was created using HTML, CSS, and Vanilla Javascript. The images display are part of focus meditations in which you have to stare at the center of the screen while you breathe.
